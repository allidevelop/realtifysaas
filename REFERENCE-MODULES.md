## Состав каждого модуля (публично)

### 1. Геопортал — ✅ есть
- Картограмма по АТЕ: 5-уровневая цветовая шкала + легенда, полноэкран, поиск по адресу.
- **Переключатель КОАТУУ ↔ КАТОТТГ** (5 уровней: область/район/громада/нас.пункт/мікрорайон).
- Фильтры: период, метрика, операция (продаж/оренда), сегмент/тип/площадь, валюта.
- Население (Держстат) в режиме КАТОТТГ. Экспорт PDF. Дву­язычие (укр/англ).
- Масштаб: **12 700+ полигонов, 430 показників на дату** (376 мікрорайонів Києва).
- Данные: собственные агрегаты ГІС «Увекон» + КОАТУУ/КАТОТТГ + Держстат (население).
- **У нас:** choropleth есть; нет тоггла КОАТУУ/КАТОТТГ, нет 430 метрик, населения, дву­язычия.

### 2. АРМ Аналітика — ✅ есть
- Доступ к **базе объявлений**; выбор по территории/адресу; фильтр/сортировка таблицы; средние; **экспорт в MS Excel**.
- Горизонт: текущая дата или ретроспектива **с января 2018**. Бандл с Калькулятором (общий пул запросов).
- Данные: объявления (не нотариальные сделки). 1 запрос = 1 массив объявлений по фильтру.
- **У нас:** поиск + CSV-экспорт есть. Близко; экспорт у нас CSV (не xlsx), общий пул с калькулятором не реализован.

### 3. Інтерактивний звіт — ✅ есть (заметно проще)
- Сквозная аналитика: средние/медианные цены, объёмы, **ставки капіталізації** (поквартально), **індекси торгу** (помісячно),
  **періоди експозиції**, **% заповнюваності комерційних площ**, **моніторинг первинки (забудовники + іпотечні програми)**,
  демографія/макро. **5 рівнів агрегації.**
- **Данные (публично заявлено именно для этого модуля): дані МінЮст про кількість нотаріальних угод** по отчуждению,
  КАТОТТГ/КОАТУУ, первинний ринок забудовників, Держстат.
- Тарифы-периоды: Тестовий (2021), Базовий (поточний+попередній рік), Корпоративний (з січня 2018).
- **У нас:** дашборды трендов цены/м² есть; **нет** cap-rate, индекса торга, экспозиции, occupancy, первички, данных МінЮст, period-tiers.

### 4. Генератор звітів — ✅ есть
- Статистический PDF-отчёт средних цен; период месяц/квартал/год с 2018; гео-уровни; сегментация. Данные: агрегаты Увекон.
- **У нас:** генерация PDF есть. Совпадает по сути.

### 5. Експрес оцінка — ✅ есть
- Мгновенная оценка/ставка аренды, текущая или ретро **с 2018**, **за Нацстандартами оцінки**, **15–30 сек, есть API**.
- Конвейер: ввод характеристик → авто-подбор аналогов → авто-корректировки → средневзвешенная.
- **У нас:** сравнительная оценка есть, ровно этот конвейер. Нет публичного API, нет ретро-выбора даты.

### 6. Калькулятор оцінювача — ✅ есть (проще)
- Авто-оценка ИЛИ **верифікація звіту про оцінку**; квартиры/коммерция/земля; текущая/ретро дата.
- Метод: сравнительный + **мінімум 200 аналогів** + **Empirical Bayesian kriging** для малоактивных регионов.
- Выходы: **шаблони звітів про оцінку (Word/PDF)** + **висновок про верифікацію**.
- Данные: Увекон + объявления агентств + партнёрские данные Асоціації спеціалістів з нерухомості.
- **У нас:** калькулятор-оценка есть; **нет** верификации отчётов, kriging, генерации Word-шаблонов оценки.

### 7. Портфельна оцінка — ❌ НЕТ у нас
- Массовая оценка: **загрузка массива в MS Excel** → авто-геокодинг каждого → подбор аналогов (ретро до 2018) → корректировки → средневзвешенная по всему массиву. Для однородных групп; авто-верификация.

### 8. Дослідження ринків / Щоквартальні звіти — ❌ НЕТ (но модель готова)
- Разовые PDF по городам (~25 стор., Power BI, 800₴): жильё/нежильё/земля ИЖС, цены $/м², по адмінрайонам, операции, торг, экспозиция, cap-rate. Данные: Увекон + **авто-сбор с 130+ сайтов агентств**.

### 9. Бібліотека професіоналів — ❌ НЕТ
- Каталог учебных пособий по оценке, поштучная продажа (email-ссылка). 5 книг (350₴ и т.п.).

## Итог: чего не хватает до референса

1. **Портфельна оцінка** (batch-оценка из Excel) — отдельный модуль, у нас нет.
2. **Дослідження ринків** (квартальные PDF по городам, 800₴) + **Бібліотека** — разовые продажи; в нашей модели `Order.refType` уже есть `research`/`library`, движка нет.
3. **Інтерактивний звіт** — догнать по составу: ставки капитализации, индекс торга, сроки экспозиции, % заполняемости коммерции, первичка/застройщики, данные МінЮст, period-tiers (2021 / тек+пред / с 2018).
4. **Калькулятор** — верификация отчётов, Word-шаблоны оценки, kriging.
5. **Геопортал** — тоггл КОАТУУ/КАТОТТГ, население, больше метрик, дву­язычие.
6. **АРМ** — общий пул запросов с калькулятором, экспорт в xlsx.
